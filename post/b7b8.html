面对对象三大特征之继承

发表于2023-03-08|更新于2023-03-08
|阅读量:

继承——面对对象三大特征之一

当类与类之间，存在相同（共性）的内容，并满足子类是父类中的一种，就可以考虑使用继承来优化代码。

——————————————————————————————————————————————————————————————————
可以将多个子类中重复的代码抽取到父类中，提高了代码的复用性。

子类可在父类的基础上，增加其他的功能，使子类更加强大。

子类可获得父类的属性和行为，子类可以使用。
——————————————————————————————————————————————————————————————————
Java 提供和 extends 关键字，使用这个关键字可以使一个类与另一个类建立起继承关系。

例：public class Student extends Person {}

继承的格式：public class 子类 extends 父类 {}

注：其中Student类被称为子类（派生类），Person类被称为父类（基类或超类）。

——————————————————————————————————————————————————————————————————

使用他人
——————————————————————————————————————————————————————————————————
特点：Java 只支持单继承，不支持多继承，但支持多层继承。

单继承：一个子类只能继承一个父类。

不支持多继承：子类不能同时继承多个父类。

多层继承：子类A继承父类B，父类B可以继承父类C。
（父类B是子类A的直接父类，那么父类C就为子类A的间接父类）

每一个类都直接或者间接的继承与Object；

——————————————————————————————————————————————————————————————————
子类能继承父类中的那些内容？

父类中有构造方法，成员变量，成员方法，均分为非私有（public）与私有（private）；

构造方法：
无论什么样的修饰符，子类都无法继承下来。

成员变量：
不管是什么修饰符，子类都可以将成员变量继承下来。（私有的可以继承下来，但是无法使用）

成员方法：
非私有的成员方法可以被继承下来，私有的不能被继承下来。
只有父类中的虚方法才可以被子类继承。
注：JVM 将会从最顶端的父类开始将会被经常用到的方法（非 private，非 static，非 final 修饰的，即虚方法。）
单独抽取出来放到所设立的虚方法表内，然后向下传递，到下一个父类会在该虚方法表的基础上添加自己类的虚方法，
以此类推，大大提高了程序的性能。

——————————————————————————————————————————————————————————————————
成员变量访问特点：就近原则，谁离我近，我就用谁。
首先会在局部位置找，本类成员位置找，父类成员位置找，逐级往上。
this 修饰：从本类成员位置开始往上找。
super 修饰：在父类成员位置开始往上找。

this：理解为一个变量，表示当前方法调用者的地址值。就是一个局部变量（为当前类）；

super：代表父类存储空间。

成员方法的访问特点：就近原则。
可通过 this 修饰来调用本类中的成员方法，通过 super 修饰来调用父类中的成员方法。
注：
方法重写
当父类的方法不能满足子类现在的需求时，需要进行方法的重写。
在继承体系中，子类出现了和父类中一摸一样的方法声明，我们就称子类这个方法是重写的方法。
并且要使用 @Override 重写注解。
1.@Override 是放在重写的方法上，校验子类重写时语法是否正确。 2.加上注解后如果有红色波浪线，表示语法错误。 3.建议重写方法都加@Override 注解，代码安全，优雅！
方法重写的本质：
子类中重写的方法会将从父类中所继承的虚方法表中的方法覆盖。
注：
重写方法的名称、形参列表必须与父类中的一致。
子类重写父类方法时，访问权限子类必须大于等于父类（暂时了解：空着不写< protected < public）
子类重写父类方法时，返回值类型子类必须小于等于父类。
建议：重写的方法尽量和父类保持一致。
只有被添加到虚方法表中的方法才能被重写。

构造方法的访问特点： 1.父类中的构造方法是不会被子类继承 2.子类中所有的构造方法默认先 访问父类中的无参构造，再执行自己。
Why？
（1）子类在初始化的时候，有可能会使用父类中的数据，如果父类没有完成初始化，子类将无法使用父类的数据。
（2）子类初始化之前，一定要调用父类构造方法先完成父类数据空间的初始化。 3.子类构造方法的第一行语句默认都是：super（），不写也存在，且必须在第一行。 4.如果想调用父类有参构造，必须手写 super 进行调用。
